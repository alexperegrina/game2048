# FIB-AS Arquitectura del Software

El proyecto consiste en la implementación del juego 2048. Nos dan un enunciado con una primera 
fase del diagrama de clases los casos de uso y los contratos de la capa de dominio. 
Para la realización del proyecto aplicamos un modelo de tres capas, se realizan cuatro entregas una por cada capa mas la 
implementación. 

El lenguaje de programación indicado es Java con una base de datos PostgreSQL, para la capa de datos utilizamos Hibernate que 
es un ORM para Java.

Las tareas a realizar son el diseño de la capa de dominio con los diagramas de sequencia y una primera implementación de dos 
clases y su relación con Hibernate.

En la especificación del proyecto tenemos que asegurar que haya:
- Nivel bajo de acoplamiento
- Alto nivel de cohesión
- Open Closed Principle (OCP)
- Dependency Inversion Principle (DIP)
- Interface Segregation Principle (ISP)
